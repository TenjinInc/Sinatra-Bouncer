#Bouncer
Simple authorization permissions plug for Sinatra. 

## Prerequisites
Bouncer requires [Sinatra](http://www.sinatrarb.com/), and [ruby 1.9.3](https://www.ruby-lang.org/en/documentation/installation/).  

## Installation

###Gemfile
```
gem 'bouncer'
```

### Command Line
```
gem install bouncer
```

##Usage
```
include Bouncer
```
